Claude Code Plugin

IM HTML Style Guide

Install guide for the Integral Media brand plugin. Once installed, Claude Code automatically uses our colours, fonts, logo, and layout when creating HTML documents.

Install Auto-Update Usage What's Included
01

Install

Important: All commands below are run from your regular terminal (bash/zsh), NOT from inside Claude Code's ❯ prompt. If Claude Code is open, exit it first.
1

Add the marketplace

This tells Claude Code where to find the plugin.

claude plugin marketplace add integralmedia1/im-html-style-guide

You should see: Successfully added marketplace: im-html-style-guide

2

Install the plugin

This downloads the style guide, template, and brand assets.

claude plugin install im-html-style-guide

You should see: Successfully installed plugin: im-html-style-guide

3

Restart Claude Code

Open Claude Code as normal. The plugin loads automatically on startup - no extra steps needed.

02

Auto-Update

Enable auto-updates (recommended)

By default, third-party plugins don't auto-update. Enable it so you always get the latest version when Claude Code starts.

1

Open the plugin manager

Inside Claude Code, type:

/plugin
2

Go to the Marketplaces tab

Select im-html-style-guide from the list.

3

Enable auto-update

Toggle it on. From now on, the plugin refreshes automatically every time you start Claude Code.

Manual update: If you prefer not to auto-update, you can pull the latest any time from your terminal: claude plugin marketplace update im-html-style-guide
03

Usage

Just ask Claude

Once installed, simply ask Claude Code to create an HTML document. It will automatically use the brand template. Examples:

  • "Create an HTML report for [client name]"
  • "Make a branded document about our SEO results"
  • "Build an IM presentation for the quarterly review"

What Claude does automatically

  • Uses the IM brand colours (Sky Blue, Deep Navy, Charcoal)
  • Applies correct typography (Julius Sans One, Inter, JetBrains Mono)
  • Inlines the IM logo as SVG (no external images)
  • Uses Lucide inline SVG icons (no emoji)
  • Includes responsive CSS for mobile devices
  • Creates fully self-contained HTML (no external stylesheets)
White-label: For Higher Ranking or Purpose Digital client documents, tell Claude it's for a white-label partner. It will automatically strip all IM logos and references.
